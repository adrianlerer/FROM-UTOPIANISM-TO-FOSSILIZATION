# Contributing to From Utopianism to Fossilization

Thank you for your interest in contributing to this research project!

## How to Contribute

### 1. Report Issues
Found an error in calculations, data, or methodology?
- Open a GitHub issue with detailed description
- Include steps to reproduce if applicable
- Suggest corrections if you have them

### 2. Improve Code
Want to enhance the analysis scripts?
- Fork the repository
- Create a feature branch (`git checkout -b feature/improvement-name`)
- Make your changes with clear commit messages
- Ensure all scripts still run successfully
- Submit a pull request with description of changes

### 3. Add New Cases
Have data for additional constitutional cases?
- Ensure data meets Reality Filter standards (see DOCUMENTATION/)
- Add analysis script following existing format
- Include CSV outputs in DATA/analysis_results/
- Document data sources thoroughly

### 4. Extend Methodology
Ideas for methodological improvements?
- Open an issue first to discuss the approach
- If approved, implement following EPT framework
- Validate against existing cases (Colombia, Chile, Argentina)
- Update documentation accordingly

## Code Standards

- **Python**: PEP 8 style guide
- **Documentation**: Clear docstrings for all functions
- **Reality Filter**: Tag all data with verification level
  - [Verificado]: Official sources
  - [Estimación]: Calculated from verified inputs
  - [Inferencia]: Logical derivation
  - [Proyección]: Counterfactual scenarios
- **Replication**: All analysis must be reproducible

## Academic Integrity

- Cite all sources properly
- Respect Dixon & Landau's original framework
- Acknowledge contributions from others
- Maintain transparency about data limitations

## Questions?

Open an issue or contact the author through GitHub.
